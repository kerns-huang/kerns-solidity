# 函数修饰符

## public 
公开函数，内部，外部都可以调用。
## external：只能外部调用，相对于public 节省gas

可以从calldata读取参数，避免内存复制，显著节省gas消耗

## internal 
类似java里面的default，子类可以访问

## private： 

只能合约内部调用，子类不可以访问

# 状态修饰符

## view ：

### 特点
只能读取变量，不能修改变量

## pure： 存数学计算和逻辑

### 特点
1. 不能读取状态变量
2. 不能读取msg.value,block等全局变量
3. 只能使用函数参数
4. 只能使用局部变量

### gas消耗

外部调用，不消耗gas
内部调用，消耗gas

## payable

### 特点
可接受ETH的一个修饰符


# 自定义modify

类似拦截器，可以有多个拦截器
函数执行前的检查点，用于在函数执行前插入特定的逻辑，例如权限控制和状态检查。

```
modify 方法(参数){

require(条件，“提示”)  
_;  

}
```
## 作用的范围

权限控制
参数娇艳
